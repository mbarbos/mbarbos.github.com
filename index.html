The Variant Light template

This is Variant Light, a free website template by Andreas Viklund. It is written using standards-compliant XHTML 1.0 Strict and CSS3, and it can serve as a starting point for building your own website. Use it and modify it as wanted and needed.

For more free templates, web design tutorials and related material, visit andreasviklund.com.
